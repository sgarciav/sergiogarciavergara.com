# Initial Setup

## Dependencies

ROS dependencies:

``` bash
$ sudo apt install ros-$release-ros-numpy
```

For Python packages, execute:

``` python
$ pip3 install PKG_NAME
```

The following are the names of the packages that you'll need to install:

1. torchvision
2. ipdb
3. numba
4. shapely
5. scipy
